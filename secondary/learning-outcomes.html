Learning outcomes

Knowledge

  • Student knows the central role that the web and the internet have in modern data processing
  • Student is familiar with relevant programming tools and repositories with version control
  • Student has basic knowledge of and experience with collaboration in software development as a form of work

Skills

  • Student can create user-friendly and universally designed solutions in line with legislation
  • Student build webpages with the key technologies for web (HTML, CSS)
  • Student can use current development tools and version control
  • Student can plan, organize and implement smaller IT projects

General competence

  • Student can communicate the result of a development work
  • Student can collaborate in groups
